Algorithms

Output Algorithm(Algorithm 1)

Baseline Exact Flow-Based Method

Our first implementation follows the baseline exact approach for finding the densest h-clique subgraph, adapting Goldberg's classic flow-based method. This algorithm relies on a binary search over the possible range of h-clique densities. For each candidate density α, it constructs a flow network to determine if any subgraph D exists such that ρh​(D)>α. The process begins by enumerating all distinct h-cliques present in the entire input graph G. This is a prerequisite for calculating vertex clique-degrees and constructing the flow network accurately. Graph information: Nodes: 1486 Edges: 3297 Directed: No

CoreExact Algorithm(Algorithm 4)

Core-Based Exact Method

To address the efficiency limitations, we implemented an optimized exact algorithm based on h-clique core decomposition. Prunes the search space using core decomposition before applying the flow-based search. Computes the h-clique core number for every vertex based on its participation in h-cliques within iteratively refined subgraphs. Identifies the maximum core number, kmax​.
